# MOTHER AI/ML Ovarian Follicle classification.
## Part of the MOTHER Project

### Program 3

This program takes as input a set of sub-images of follicles, organized in folders by follicle types, orgainized in Train Test and Validate subsets and does transfer learning from the ResNet34 trained on ImageNet data.

The final trained ovary image ResNet network is in the file ```ResNet34_Small_Follicles_Epoch_24.pt``` (83MB).
